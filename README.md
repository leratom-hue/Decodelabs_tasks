# DecodeLabs Data Analytics Internship Project

## About the Project

This repository contains the final project I completed during the DecodeLabs Data Analytics Internship. The project gave me the opportunity to work through the basic stages of a real data analytics workflow, starting from messy raw data and ending with a dashboard that communicates insights clearly.

The goal of the project was not only to analyse data but also to understand how important data preparation is before any analysis takes place. I worked with the dataset using Microsoft Excel to clean and organise the data, and then used Microsoft Power BI to explore patterns and present the results visually.

This project helped me practice thinking like a data analyst by focusing on data quality, identifying patterns, and explaining what the data shows.

---

## Tools Used

For this project I used the following tools:

* Microsoft Power Bi (Power Query) – for cleaning and preparing the dataset
* Microsoft Power BI – for analysing the data and building visualisations
* GitHub – to document and share the project

---

## Data Cleaning

The first part of the project focused on preparing the dataset. The raw data contained several issues that needed to be addressed before any analysis could be done.

During this stage I checked the dataset carefully and corrected problems such as duplicate records, missing values, and inconsistent formatting. I also standardised column names and ensured that dates and numbers were stored in the correct format. Cleaning the data made the dataset more reliable and easier to analyse.

Working through this step showed me how much time analysts actually spend preparing data before generating insights.

---

## Data Analysis

Once the dataset was cleaned, I explored the data using basic statistical measures such as the mean, median, and count. These calculations helped me understand the overall distribution of the data and the typical values within the dataset.

Looking at the numbers also helped highlight unusual values that could potentially be outliers. I then explored the data further to see whether there were any patterns or trends that could explain those differences.

---

## Data Visualisation

After completing the analysis, I imported the dataset into Microsoft Power BI to build an interactive dashboard. The aim of the dashboard was to make the information easier to understand and allow users to explore the data visually.

The dashboard includes visualisations that show trends over time, comparisons between different categories, and key summary metrics. Creating these visuals helped me see how data can be communicated in a way that is easier for decision-makers to understand.

---

## What I Learned

Working on this project helped me strengthen several important data analytics skills. I gained more experience with data cleaning, basic statistical analysis, and building dashboards that highlight meaningful insights.

It also showed me how important it is to organise and validate data before trying to interpret it. Even simple analysis can lead to misleading results if the underlying data is not properly prepared.

---

## Conclusion

Overall, this project helped me understand the full process of turning raw data into useful information. Starting with messy data and ending with a structured dashboard helped me see how each stage of the analytics process connects to the next.

This internship project was a valuable learning experience and helped me build practical skills that I can continue developing in future data analytics projects.
